* tag version in git

  $ git tag -a v0.3.23

* write release notes.  git shortlog helps a lot:

  $ git shortlog --no-merges v0.3.22..

* create source-only archive

  $ git archive --format=tar --prefix=cosbycoin-0.3.23/ HEAD | \
  	gzip -9c > ~/tmp/cosbycoin-0.3.23-src.tar.gz 

* perform gitian builds

  * From the cosbycoin source dir
  $ cd ../gitian-builder
  $ ./bin/gbuild --commit cosbycoin=v0.3.23 ../cosbycoin/contrib/gitian.yml
  $ ./bin/gbuild --commit cosbycoin=v0.3.23 ../cosbycoin/contrib/gitian-win32.yml
 
  Build output expected:
  1. linux 32-bit and 64-bit binaries + source
  2. windows 32-bit binary + source
